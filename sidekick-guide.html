GPI Sidekick For Docs Guide

1. Sync with sheet

To sync with sheet, you should have atleast two tables.

  • First table with containing data of a single row in sheet.
  • The heading of the data represent the ID row name in the sheet.
  • Second table containing the metadata of the sheet. It contain spreadsheet link and sheet name.

GPI Sidekick for sheets guide

1. Sync with doc

To sync with doc, you should have atleast two columns.

  • First Column with the heading of ID.
  • Second column containing some data with some heading.
  • Then at the end we have "attached_single_doc" and/or "attached_doc" column in the end. They represent attached doc for full sheet or attached doc for every row resspectively. If they does not exist already, you can create them.

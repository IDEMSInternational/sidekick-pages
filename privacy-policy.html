Privacy policy

This Privacy Policy describes how the extension accesses, uses, stores, or shares Google user data. The GPI Sidekick Extension is designed to assist the Global Parenting Initiative (GPI) team with their development tasks, and all data accessed by the extension is used solely for this purpose.

Access to Google User Data:

The GPI Sidekick Extension requires access to the following Google user data: Google Sheets and Google Drive. The extension requires these permissions to create and manage functionality of the sheets, docs, and images. The extension only accesses the data required for its functioning, and it does not access any data without the user's permission.

Use of Google User Data:

The GPI Sidekick Extension uses Google user data to create and manage templates and feeds for the GPI team. The extension does not use any data for any other purpose, and it does not share any data with any third party.

Storage of Google User Data:

The GPI Sidekick Extension stores Google user data in Google Sheets and Google Drive. All data is stored securely in the user's account, and the extension does not store any data on any other server.

Privacy Policy Requirements:

The GPI Sidekick Extension privacy policy is hosted on the verified domain of the GPI website, is visible to users, and is linked on the app home page. This privacy policy clearly describes the way the GPI Sidekick Extension accesses, uses, stores, or shares Google user data.

By using the GPI Sidekick Extension, you agree to the terms of this Privacy Policy. If you do not agree with this Privacy Policy, please do not use the GPI Sidekick Extension.

Limited Use Requirements

This Google App Script extension is designed to assist the Global Parenting Initiative (GPI) team with their development tasks. Our app follows the Google API Services User Data Policy, including the Limited Use requirements. For more information, please refer to the Google API Services User Data Policy link in our app. The GPI Sidekick extension has several key features that help users create and manage their sheets and docs conveniently. The extension allows users to create different types of sheets using a custom formatting template, including Global Sheets, Template Sheets, and Database Sheets. It also includes a GitHub action feature, allowing users to push images to a GitHub repository from their Google Drive without using GitHub. In addition, the GPI Sidekick extension has a Doc feature that enables users to create a Google Doc for each row available in a sheet. Users can initialize a Doc for each row and sync it with the Google Sheet to update any changes made in the sheet and also vice versa. Overall, the GPI Sidekick extension provides a range of useful features to help the GPI team manage their templates and feeds easily and efficiently.
